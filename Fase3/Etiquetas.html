Son varias, y cada una tiene una utilidad diferente la cual es necessaria saber para utilizarla. Y se muestran a continuación:

-Para empezar, debemos poner la primera que es HTML.



Luego pondremos HEAD, que significa cuerpo la cual nos dejara poner el nombre a la pagina el color y como sera el diseño del texto.


Para poner un titulo utilizaremos TITTLE, y su tamaño varia utilizando H1, H2, H3.


Para agregar color a una pagina el codigo es BODY BGCOLOR, mas la referencia del color que queramos.


Despues de utilizar estas herramientas para el principio empezaremos con el cuerpo y para ello utilizaremos la etiqueta BODY; en la cual podremos poner texto, imagenes, cuadros y demas cosas que necesitemos en nuestra pagina.


Para el cuerpo de nuestra pagina, utilizaremos las siguientes etiquetas:

P: Parrafo


ARTICLE: Articulo


SECTION: Sección


BR: Salto de línea


TABLE: Insertar tabla


IMG: Imagén.


Para terminar nuestra página, debemos terminar con FOOTER, para poder poner las referencias bibliograficas de donde hemos sacado nuestra información.


Nota: debemos tener en cuenta que cada vez que agregamos una etiqueta debemos cerrarla, en el mismo orden que las ponemos.
